1. Убедиться, что используем AHCI
2. Убедиться, что используем GPT
3. (Опционально) Очистить диск от разделов

Если используем
4.1. Установить указав букву системного диска Z

Если обычная установка
4.1. Устанавливаем
4.2. Перезагружаем
4.3. Меняем букву используя Acronis

5. Отключаем индексацию на системном разделе
6.1. Перезагрузка
6.2. (Win11) IMPORTANT!!! Disable 'Secure Boot' option in bios if you dont want to encrypt all your drives. By default windows encrypts all ntfs volumes on first install, including your data volumes.
7. В начале все по умолчанию
8. На этапе создания пользователя выбираем "Присоедениться к домену"
9. Создаем учетную запись с именем английскими символами без пробелов. Если нужен пароль, то лучше установить из системы, чтобы не задавать вопросы для восстановления

10. Установливаем обновления до тех пор пока обновления не будут найдены, после перезагружаем
11. Отключаем автозапуск подлюченных устройств Настройки -> Устройства -> Автозапуск -> Выключить (Win11: Settings -> Bluetooth & Devices -> AutoPlay)

12.0. Меняем на большие значки в "панели управления"

12.1. Ставим переключение раскладки на ctrl+shift Настройки -> Устройства -> Ввод -> Дополнительные настройки -> Горячие клавиши (Win11: Settings -> Time & Languages -> Typing -> Advanced Keyboard Settings -> Input language hot keys)
12.2. Ставим разные языки для каждого окна Настройки -> Устройства -> Ввод -> Дополнительные настройки -> Позволить использовать разные языки ввода для каждого окна
12.3. Отключаем ускорение курсора Панель управления -> Мышь -> Параметры указателя -> Убираем галочку с повысить точность

13.1. Закрепить "панель управления" на начальном экране
13.2. Закрепить "этот компьютер" на начальном экране

14.1. Перемещаем системную temp папку в корень на системном диске Этот компьютер -> Конеткстное меню -> Свойства -> Дополнительные системные настройки -> Переменные окружения -> Системные переменные -> Temp
14.2. (Опционально) Перемещаем пользовательскую temp папку в корень на системном диске Этот компьютер -> Конеткстное меню -> Свойства -> Дополнительные системные настройки -> Переменные окружения -> Пользовательские переменные -> Temp

16. Отключаем uac Панель управления -> Пользователи -> Изменить UAC -> Ставит на минимум
17. (Опционально, Не рекомендуется) Отключаем брандмауэр Панель управления -> Брандмауэр -> Отключить
18. Ставим начальную вкладку в проводнике на этот компьютер Этот компьютер -> Вид -> Опции -> Открывать с Этот компьютер (Win11: Этот компьютер -> ... -> Properties ...)
19. Ставим настройки отображения в проводнике Этот компьютер -> Вид -> Опции -> Вид -> Показывать скрытые файлы и Показывать полный путь в строке и Скрывать пустые диски и Скрывать расширения для зарегестрированных типов
20. Отключаем защитник Защитник -> Защита от вирусов и угроз -> Управление настройками -> Отключаем всё и запускаем http://download.ufeff.xyz/Windows10Improvements/DisableDefender.reg
21. Перезагрузить
22. (Only Win10) Убираем из раздела этот компьютер в проводнике ненужные элементы /Registry/FileExplorer64bit
23.1. Отключаем ненужное отслеживание действий Настройки -> Конфиденциальность
23.2. Disable Windows Recall / Windows AI. Command to disable: `DISM /Online /Disable-Feature /Featurename:Recall`. Command to check: `DISM /Online /Get-Featureinfo /Featurename:Recall`.
23.3. Disable Windows Security -> Core Isolation features
24.1. (LEGACY!!!) ~~~Поставить oem ключ через kms auto http://download.ufeff.xyz/Windows10Improvements/KMSAutoNet_1.5.4/~~~
24.2. Install license using https://github.com/massgravel/Microsoft-Activation-Scripts. Or just paste this command into PowerShell `irm https://get.activated.win | iex`. Use HWID by default. For Win11 LTSC use KMS 180 days
25. Делаем файл подкачки фиксированным Этот компьютер -> Контекстное меню -> Свойства -> Дополнительные системные настройки -> Производительность -> Настройки -> Дополнительные -> Виртуальная память -> Изменить -> Выбирать автоматически -> Свой размер -> 8192 в оба поля -> Установить
26. Отключаем быструю загрузку Панель управления -> Электропитание -> Выбрать что делают кнопки питания -> Изменить настройки которые недоступны -> Снять галочку Быстрая загрузка
27. Отключить бэкапы на системном диске Этот компьютер -> Контекстное меню -> Свойства -> Дополнительные системные настройки -> Защита системы -> Конфигурация -> Отключить защиту и Максимальное использование в 0 и Удалить
28. (Опционально) Изменяем имя ПК Этот компьютер -> Контекстное меню -> Свойства -> Дополнительные системные настройки -> Имя компьютера -> Изменить

29. Ставим маленький значок поиска Панель задач -> Поиск -> Показывать иконку
30.1. Настройка панели задач Персонализация -> Панель задач -> Использовать маленькие иконки и Комбинировать кнопки если панель задач полная
30.2. (Only Win11) Moving task bar to left side Personalization -> Taskbar  Behaviors -> Taskbar alignment -> Left

31.1. Устанавливаем часовой пояс Дата и время -> Настройки даты и времени -> Дата и время -> Часовой пояс
31.2. (Опционально) Устанавливаем европейский формат чисел Дата и время -> Настройки даты и времени -> Региональный формат -> English (Europe)
31.3. (Опционально) Установить русскую раскладку Дата и время -> Настройки даты и времени -> Язык -> Добавить язык -> Русский

32. Делаем поверх всех окон диспетчер задач Панель задач -> Контекстное меню -> Диспетчер задач -> Опции -> Поверх остальных окон

33. Перезагружаем

34.1. Установить 7-zip https://7-zip.org/
34.2. (Optional) Installing Microsoft Store On Win10 LTSC use https://github.com/kkkgo/LTSC-Add-MicrosoftStore. On Win11 LTSC just insert command into powershell `wsreset -i`

35. Устанавливаем драйвера Чипсет, Видеокарта, Сеть, Звук и другое. Можно ручками, можно через SDI (https://sdi-tool.org/). Исключением, то есть лучше установить ручками, являются драйвера на GPU, так как автоустановщики зачастую не устанавливают, например, в случае с nVidia, то самое приложение для управления и для записи
36. Перезагружаем

37. (Опционально) Меняем план электропитания на ультра (Зачасту влияет на систему незначительно, но все же)
